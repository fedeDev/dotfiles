# Personal Configuration and Useful Tools

Setup (assuming ubuntu-based distro):
```bash
sudo apt install git
git clone <repo> ~/dotfiles
cd ~/dotfiles
stow <each folder>
```
Compile st and register it with ```tic -s st.info```.
